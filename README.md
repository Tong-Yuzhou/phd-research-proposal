# PhD Research Proposal

This repository contains the LaTeX source files for my PhD research proposal.

## Main Files

- `main.tex`: main LaTeX file
- `sections/`: proposal section files
- `references.bib`: bibliography file
- `proposal.pdf`: compiled PDF version

## Compilation

The proposal can be compiled in Overleaf using `main.tex`.

Additional figures or supporting data will be added in later stages if required.
